How to use:
1. Open index.html in a browser.
2. Upload the whole folder to Netlify / Vercel / GitHub Pages.

Features:
- 3 languages: English, Hindi, Marathi
- Theme modes: System, Light, Dark, Extra Dim
- Mobile-friendly layout
- Call / WhatsApp / Email / Maps buttons
- Uses your uploaded logo and clinic images
